The Rust Programming Language

常见的编程概念

这一章涵盖了几乎每种编程语言都会出现的概念以及它们在 Rust 中的工作方式。许多编程语言在其核心部分有很多共同之处。本章介绍的概念都不是 Rust 独有的，但我们将讨论这些概念在 Rust 中的使用，并解释使用这些概念的约定。

具体来说，您将学习变量、基本类型、函数、注释和控制流。这些基础将在每个 Rust 程序中出现，早期学习它们将为您打下坚实的基础。

关键字

Rust 语言有一组仅由语言本身保留使用的 关键字，这与其他语言类似。请注意，您不能将这些词用作变量或函数的名称。大多数关键字都有特殊含义，您将在 Rust 程序中使用它们来完成各种任务；少数关键字目前没有与之关联的功能，但已为将来可能添加到 Rust 的功能预留。您可以在 附录 A 中找到关键字的列表。
